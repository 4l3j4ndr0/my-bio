## Install the dependencies


```
npm install
```

## Run DEV environment

```
npm run dev
```

## Install AWS Amplify sandbox environment

```
npx ampx sandbox
```

or in case that you have AWS profile configured

```
npx ampx sandbox --profile <your-aws-profile>
```
